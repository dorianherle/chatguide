# ChatGuide Streamlit App

A Streamlit web interface for the ChatGuide conversational AI framework.

## Setup

1. Clone this repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Create a `.env` file with your API key:
   ```
   GEMINI_API_KEY=your_gemini_api_key_here
   ```
4. Run the app:
   ```bash
   streamlit run streamlit_app.py
   ```

## Features

- Multi-language support
- Progressive conversation flow
- Real-time state tracking
- Chat export functionality
- Password protection

## Dependencies

This app uses the ChatGuide package from GitHub. The package will be automatically installed when you run `pip install -r requirements.txt`.
